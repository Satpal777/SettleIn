# Admin Workflow

![SettleIn Logo](./images/flow (1).png)

The Admin workflow ensures system integrity, user trust, and overall platform oversight.

## 1. System Dashboard
*   **Overview**: Monitor platform activity stats (Total Users, Active Listings, Completed Move-ins).
*   **Recent Activity**: View a feed of significant events across the platform.

## 2. User Management
*   **Approval**: Review new user sign-ups and profiles.
*   **Role Management**: Update user roles (Seeker, Landlord, Admin).
*   **Moderation**: Disable or flag users who violate platform policies.

## 3. Listing Oversight
*   **Verification**: Review new property listings for accuracy and compliance.
*   **Management**: Edit or remove fraudulent or outdated listings.
*   **Status Review**: Monitor available vs rented property ratios.

## 4. Support & Maintenance
*   **Ticketing Overview**: Access a global view of all support and maintenance tickets.
*   **Intervention**: Resolve disputes or technical issues between seekers and landlords.
